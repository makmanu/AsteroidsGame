# AsteroidsGame #
## Description ##
Asteroid game that runs on pygame
## Run ##
Write commands in root folder
1. `python3 -m venv venv`
2. `source venv/bin/activate`
3. `pip install -r requirements.txt`
4. `python3 main.py`
## Requirements ##
Pygame 2.6.1
## Controls ##
WASD - move   
spacebar - shoot
